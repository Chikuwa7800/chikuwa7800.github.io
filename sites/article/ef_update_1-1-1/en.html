Elder Fire Force 1.0.1 Update Was Released!

New Feature

Added an advancement of Bazooka.

Changes

Now compatible with Minecraft 26.2.

The guns reloading method has been changed.

Changing the gun reloading method

  • Reloading can now be initiated by default using Left Ctrl + Left Shift + Right-Click when the weapon is unloaded and not in a ready-to-fire state.
  • It is now possible to transition to a ready-to-fire state even when unloaded. (Note: Firing is not possible as no ammunition is loaded.)
  • This enhances immersion.
