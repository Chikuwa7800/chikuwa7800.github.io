要覧

博物館にでも飾られていそうな古い銃を追加する。追加された銃は使用に弾を消費する。ただ、弾はクラフトで大量に作れないので、慎重に使うことを推奨する。

ダウンロード

ダウンロードはGitHubからする必要がある。本データパックはModrinthに公開していない。

GitHub

GitHub Chikuwa7800/Elder_Fire_Force

追加要素

本データパックではアイテムのみ追加される。

アイテム

　　詳細は「アイテム」を参照

アイテムには、銃と弾があり、銃は弾を消費してリロードを行う。また、銃には耐久値が存在し、これが0になると使用不可になる。
